# Vision

BuildTrack ist eine moderne SaaS-Webanwendung für Bauunternehmen zur Verwaltung von:

- Baustellen
- Mitarbeitern
- Arbeitszeiten
- Fotodokumentationen
- Auswertungen

Die Anwendung soll später mehrere Unternehmen (Multi-Tenant) unterstützen.
